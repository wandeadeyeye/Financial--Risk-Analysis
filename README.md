I just concluded analysing financial dataset to help and supports data-driven decision-making for credit risk teams, financial analysts, and lending institutions.

 Financial Risk Assessment and Loan Default Analysis in Power BI

Summary:

This project presents a comprehensive analysis of financial risk using Power BI, focused on loan behavior, income patterns, credit profiles, and default rates. The report integrates demographic, geographic, and financial data to uncover key risk trends and highlight high-risk segments.

Key insights include:

Demographic profiling by age, gender, education, and marital status

Geographic income and loan distribution across 243 countries

Risk classification based on credit score, loan purpose, and employment stability

Loan default analysis, including income levels, credit scores, and default frequency

Interactive dashboards using maps, risk heatmaps, and trend visuals for strategic decisions

The project demonstrates data storytelling, dashboard design, and analytical thinking to support informed credit risk management.

It supports data-driven decision-making for credit risk teams, financial analysts, and lending institutions.

To make informed lending decisions, risk mitigation, and customer segmentation based on multiple criteria.
